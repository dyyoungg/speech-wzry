# speech-wzry
爬取王者荣耀世界站的所有英雄不同皮肤的语音

首先爬取所有英雄的语音json文件，里面包含了不同皮肤的所有语音地址信息，将json文件存到本地，存储结构如下

```
—— voice_json
———— 105_廉颇.json
———— 106_小乔.json
.....

```

然后根据每个英雄语音json文件解析不同皮肤的语音地址，下载即可。存储结构如下
```
—— raw_voices
———— 亚瑟
———————— 圣骑之力
————————————因剑而亡.wav
———————— 心灵战警
———————————— xxxxx.wav
———————————— xxxx.wav
```
